# Chattingo – Full Stack Chat Application (DevOps Project)

This project is a real-time chat application built using:

- Frontend: React
- Backend: Spring Boot
- Communication: WebSockets
- Containerization: Docker
- Orchestration: Docker Compose
- CI/CD: Jenkins (Upcoming)
- Cloud Deployment: AWS (Upcoming)
- Infrastructure as Code: Terraform (Upcoming)

This project demonstrates end-to-end DevOps implementation.

## Architecture

User → React Frontend → Spring Boot Backend → WebSocket Communication
